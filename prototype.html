// 001 — Origin

The kind of builder
who reads the RFC.

Started in Java. Terminal apps. Four-class MVC-like elegance with zero persistence — because real learning happens from constraints.

Moved into AI/ML because the field was genuinely unsolved. Not chatbots. Enterprise-grade RAG with role-based access control at the vector DB layer. LangChain, Qdrant, Casbin, and Microsoft Presidio — not tutorials, actual systems.

Now building MiniVC — a Python version control tool with Myers diff from scratch, content-addressable storage, and a VS Code-inspired UI. Because reinventing the wheel with precision is how you really understand the wheel.

Based in Hyderabad. Building for the world.
